# Kebijakan Keamanan (Security Policy)

Kami mengutamakan keamanan dan integritas data dari landing page **Zenoria MC** serta keselamatan pengguna akhir (pemain). Jika Anda menemukan celah keamanan (vulnerability) pada website ini, harap laporkan kepada kami secara bertanggung jawab.

## Melaporkan Celah Keamanan

Harap **TIDAK** melaporkan celah keamanan secara terbuka melalui sistem GitHub Issues.

Alih-alih, silakan kirimkan laporan Anda secara langsung dan tertutup melalui salah satu kontak berikut:
- **WhatsApp Pengembang (RAN DEV)**: `0895602592430`
- **Email**: `jimmyrp.personal@gmail.com` (Sesuai dengan administrator kontak utama)

Kami berjanji akan merespons laporan Anda dalam waktu maksimal **2x24 jam** dan mendiskusikan langkah penyelesaian bersama sebelum celah keamanan dipublikasikan secara umum.

## Cakupan Keamanan

Kebijakan ini berlaku untuk:
- Kode sumber frontend React, TypeScript, dan konfigurasi build.
- Kebocoran kredensial atau API Key yang tidak disengaja di repositori.
- Celah skrip lintas situs (Cross-Site Scripting / XSS).

Terima kasih atas kontribusi Anda dalam menjaga keamanan website kami!
